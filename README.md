# 【技育CAMP2024】ハッカソン Vol.7 : 作品「走れバステト」

## 概要

「走れバステト」は2024年6月23日に開催された【技育CAMP2024】ハッカソン Vol.7で努力賞を受賞した作品です。<br>pygameを用いて作成した2D横スクロールアクションゲームとなっています。
